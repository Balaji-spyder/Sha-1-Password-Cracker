How to Use the Application

GitHub link: https://github.com/Balaji-spyder/Sha-1-Password-Cracker

Welcome to the SHA-1 Password Cracker Console! This tool helps you crack SHA-1 hashes and recover passwords. Don’t worry if you’re new to this—it’s easy to use! Just follow these steps:

  1. Enter a Hash: Type or paste a valid SHA-1 hash into the input box. (A hash is a scrambled version of a password, usually a 40-character string like 5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8.)
  2. Select Salt Option: If the hash uses a salt (extra data added to the password), choose Yes from the dropdown menu. - Not sure? Leave it as No—most hashes don’t use salts.
  3. Click "Execute": Once you’ve entered the hash and selected the salt option, click the Execute button. - The system will process the hash and attempt to crack it.
  4. View the Result: If the hash is cracked successfully, the original password will appear below the button. - You can also view all cracked passwords in the table below, along with a Copy button to easily copy the result.
  5. Try Sample Hashes: If you’re just testing the tool, use sample hashes from the tests.txt file in the GitHub repository. These are pre-made hashes to help you get started.

That’s it—you’re ready to go! Feel free to experiment and explore.
